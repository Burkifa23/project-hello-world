# All Languages Starting With C

# C – ISO/IEC 9899
- **Description**: C is a general-purpose programming language that has influenced many other modern languages.
- **Author**: Dennis Ritchie
- **Paradigm**: Procedural, imperative
- **Famous Use Case**: System software, operating systems like UNIX.

---


# C-- (C minus minus)
- **Description**: C-- is a portable assembly language designed to be a compiler target.
- **Author**: Simon Peyton Jones and Norman Ramsey
- **Paradigm**: Imperative, low-level
- **Famous Use Case**: Intermediate representation for compiler design.

---


# C++ (C plus plus) – ISO/IEC 14882
- **Description**: C++ is an extension of C that includes object-oriented features.
- **Author**: Bjarne Stroustrup
- **Paradigm**: Multi-paradigm (procedural, object-oriented, generic)
- **Famous Use Case**: Game development, system software, large-scale applications.

---


# C*
- **Description**: C* (C Star) is a data-parallel superset of ANSI C.
- **Author**: Thinking Machines Corporation
- **Paradigm**: Data-parallel, imperative
- **Famous Use Case**: Parallel computing applications.

---


# C# (C sharp) – ISO/IEC 23270

- **Famous Use Case**: Windows applications, enterprise software, game development with Unity.
